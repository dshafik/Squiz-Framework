Assigning Permissions to Users and User Groups

To assign a permission to a user or user group:

  1. In the Projects section, select the project that you want to assign the permission for.
  2. Click Add New for the permission that you want to assign the user or group to.
  3. Select the user or group that you want to assign the permission to.
  4. Click Select Asset(s). The user or group will appear in the Users and Groups section for the permission.
  5. Click Save. The user or group will be assigned the permission and have access to the priviliges that have been defined for the role.
