Projects Section

This section lists the projects that have been created within the system.

Permissions Section

This section lists the roles that have been set up in the system. For each role, you can assign a user or user group so that they recieve the privileges defined for that role.

Users and Groups Section

This section allows you to assign users and user groups to a role and remove users and user groups from the list.

Add New Button

This button allows you to add a user and user group to the Users and Groups section.

Delete Icon

This icon allows you to remove a user or user group from a Users and Groups section. When you click this icon, the user or user group will remain on the screen but it will be disabled. It will be removed when you click Save. If you marked the wrong user or user group for deletion, click this icon once again.
