Removing Permissions for a User and User Group

To remove permissions for a user or user group:

  1. In the Projects section, select the project that you want to remove the permission for.
  2. Click the Delete icon for the user or group that you want to remove the permission for.
  3. Click Save. The user or group will no longer be assigned the permissions for the project.
