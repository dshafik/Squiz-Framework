Adding a Role

To add a role:

  1. Click Add New.
  2. Enter a name for the new role in the Name field.
  3. Click Create. The new role will appear in the Role Manager section.
  4. Set the privileges for the role on each of the tabs that are available by clicking the Privilege icon.
  5. Click Save. The new role will be created in the system
