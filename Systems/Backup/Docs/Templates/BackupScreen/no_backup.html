Why are there no backup files listed in the Available Backups section?

Backups of the system will only be taken when the Backup cron job has been enabled. If the cron job is not enabled, no backup files will be listed in the Available Backups section. For more information on how to enable a cron job, refer to the article

Enabling a Cron Job
