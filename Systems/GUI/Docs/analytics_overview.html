Using the Squiz Analytics interface

Once you have logged into your system, the black toolbar at the top of the page is where you can find the buttons that allow you to perform the tasks required to view the statistics of a site.

The Mode Switcher allows you to navigate to the different modes that are available in the system. These modes include:

  • Dashboard:
  • Goals:
  • Reports:
  • Test:
  • System Config:
The modes available to you in the mode switcher will depend on the level of access you have in the system.

To change which mode you are viewing, click on the name of the mode in the Mode Switcher. To view all modes that are available in the mode switcher, click the Mode Switcher Arrow icon.

The next set of icons that are available in the toolbar include the following:

  • Profile Button:
  • Logout Button:
  • Help Button:

The Save button allows you to save your changes and the Revert Changes button allows you to revert any changes that you have made.
