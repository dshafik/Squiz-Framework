tasks This article explains how to enable the notification email so that a user will receive an email when a new update becomes available.

Enabling the Notification Email

To enable the notification email for the up to date system:

  1. In the Notify of New Updates field, select yes.
  2. Click Add Users.
  3. Select the user that you want to send the email to.
  4. Click Select. The user or user group will appear in the Notification list.
  5. Click Save.
