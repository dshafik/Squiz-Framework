Checking for New Updates

By default, the system will check for updates every hour. If there are new updates available, they will be downloaded and displayed on the New Updates tab, where you can

schedule
them.

To check for new updates earlier than what is scheduled, click Check ASAP. For example, if the Up-to-Date Service cron job is set to Frequent, you can click this button to check for updates the next time the cron job is run. By default, this will be within 5 minutes.
