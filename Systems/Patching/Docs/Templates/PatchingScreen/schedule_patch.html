Scheduling New Patches for Installation

To schedule new patches for installation:

  1. Click the New Updates tab.
  2. In the Schedule Installation of Updates field, select yes.
  3. Enter the date to run the update in the Scheduled Date field.
  4. Enter the time to run the update in the Scheduled Time field.
  5. Click Save. The patches listed in the Patches section will be scheduled for installation.
