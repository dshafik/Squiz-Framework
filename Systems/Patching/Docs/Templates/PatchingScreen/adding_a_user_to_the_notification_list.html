tasks This article explains how to add a user to the notification list so that they are sent an email when a new update is available.

Adding a User to the Notification Email

To add a user to the notification email:

  1. Click Add Users. The asset finder will appear.
  2. In the asset finder, select the user that you want to send the email to.
  3. Click Select. The user will appear in the Notification list.
  4. Click Save. The user will now be sent an email when a new update becomes available.
