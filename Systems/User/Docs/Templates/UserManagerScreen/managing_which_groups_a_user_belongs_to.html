Managing which User Groups a User Belongs to

The Belongs to Groups section allows you to manage which user groups a user belongs to. Here you can add the user to another user group by clicking Add to another Group and remove the user from a user group by clicking the Delete icon.

To view the Belongs to Groups section:

  1. Select Local Users and Groups
  2. Select the user that you want to modify. The Belongs to Groups section will appear on the screen.
