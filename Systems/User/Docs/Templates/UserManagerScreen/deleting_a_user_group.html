Deleting a User Group

To delete a user group:

  1. Select Local Users and Groups.
  2. Select the user group that you want to delete.
  3. Click the Delete icon for each of the groups listed in the Belongs to Groups section. For the last group, a warning message will appear. Click Remove.

    Please note, any users that have been created in this group and have not been linked into other groups will also be deleted.
  4. Click Save. The user group will be deleted.
