Adding an Existing User to a User Group

To add an existing user to a user group:

  1. Select Local Users and Groups.
  2. Select the user group that you want to add the user to.
  3. Click Add Existing User.
  4. Select the user that you want to add.
  5. Click Select Asset. The user will be added to the selected user group.
  6. Click Save.
