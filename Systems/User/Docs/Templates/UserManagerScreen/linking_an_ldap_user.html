Adding a LDAP user into a User Group

To add a LDAP user to a user group:

  1. Select LDAP Connections.
  2. Select the LDAP user that you want to link to a user group.
  3. Click Add User to Another Group.
  4. Click Link Existing.
  5. Select the group that you want to add the user to.
  6. Click Select Asset.
  7. Click Save. The LDAP user will be added to the selected group and will recieve any permissions that have been given to that group.
