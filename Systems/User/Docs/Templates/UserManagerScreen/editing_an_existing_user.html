Editing an Existing User

To edit the details for an existing user:

  1. Select Local Users and Groups.
  2. Select the user that you want to edit.
  3. Edit the first name of the user in the First Name field.
  4. Edit the last name of the user in the Last Name field.
  5. Edit the email address of the user in the Email field.
  6. Edit the username for the account in the Username field.
  7. Click Save. The details for the user will be updated.
