Removing a User Group from another User Group

To remove a user group from another user group:

  1. Select Local Users and Groups.
  2. Select the group that you want to edit.
  3. In the Belongs to Groups section, click the Delete icon for the user group that you want to remove the selected group from.
  4. Click Save. The user will be removed from the user group.
