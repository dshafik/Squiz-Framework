Adding a New LDAP Connection

To add a new LDAP connection:

  1. Select LDAP Connections.
  2. Click New Connection. Additional fields will appear on the screen.
  3. Enter a name for the new connection into the LDAP Connection Name field.
  4. Enter the host to use to connect to the LDAP directory into the Host Name field.
  5. Enter the port to connect to the LDAP directory into the Port field.
  6. Enter the base DN of the LDAP directory into the Base DN field.
  7. Enter the DN of the user to bind as into the Bind DN field.
  8. Enter the password to use for the LDAP directory into the Password field.
  9. Enter the auth DN into the Auth DN field. This field is optional.
  10. Enter the auth filter into the Auth Filter field. This field is optional.
  11. Enter the field to use as the username into the Username field.
  12. Enter the field to use as the name of the users into the Name field.
  13. Enter the field to use as the first name of the users into the First Name field.
  14. Enter the field to use as the last name of the users into the Last Name field.
  15. Enter the field to use as the email address of the users into the Email field.
  16. Enter the field to use as the group name into the Group Name field.
  17. Enter the field to use as the group membership into the Group Membership field.
  18. Enter the field to use as the group members into the Group Members field.
  19. Enter the field to use as unique identifier for users into the Unique ID Attribute for Users field.
  20. Enter the field to use as unique identifier for groups into the Unique ID Attribute for Groups field.
  21. Click Create Connection. The LDAP connection will be created within the system.
