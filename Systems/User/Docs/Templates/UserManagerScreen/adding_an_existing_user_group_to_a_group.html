Adding an Existing User Group to a User Group

To add an existing user group to another user group:

  1. Select Local Users and Groups.
  2. Select the group that you want to add the user group to.
  3. Click Add More.
  4. Click Link Existing.
  5. Select the user group that you want to add.
  6. Click Select Asset.
  7. Click Save. The user group will be added to the selected group.
