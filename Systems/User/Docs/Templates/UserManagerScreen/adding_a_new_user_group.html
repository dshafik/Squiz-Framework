Adding a New User Group

To add a new user group:

  1. Select Local Users and Groups.
  2. Select the group that you want to add the new user group to.
  3. Click Add More.
  4. Click Create New Group.
  5. Enter the name for the user group into the Group Name field.
  6. Click Create User Group. The new user group will be created under the selected group.
