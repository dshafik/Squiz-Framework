Adding a New User Group

To add a new user group:

  1. Select Local Users and Groups.
  2. Select the user group that you want to add the new user group to.
  3. Click Add New User Group. Additional fields will appear on the screen.
  4. Enter the name for the user group into the Group Name field.
  5. Click Create User Group. The new user group will be created under the selected user group.
