Adding a New User to a User Group

To add a new user to a user group:

  1. Select Local Users and Groups.
  2. Select the group that you want to add the new user to.
  3. Click Add New User. Additional fields will appear on the screen.
  4. Enter the first name of the user into the First Name field.
  5. Enter the last name of the user into the Last Name field.
  6. Enter the email address of the user into the Email field.
  7. Enter a username for the new account into the Username field.
  8. Enter a password for the new account into the Password field.
  9. Click Create User. The new user will be created under the selected group.
