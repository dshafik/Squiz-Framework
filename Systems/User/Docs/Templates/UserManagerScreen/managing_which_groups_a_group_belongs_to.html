Managing which Groups a User Group Belongs to

The Belongs to Groups section allows you to manage which groups a user group belongs to. Here you can add the group to another user group by clicking Add to another Group and remove the group from a user group by clicking the Delete icon.

To view the Belongs to Groups section:

  1. Select Local Users and Groups
  2. Select the group that you want to modify. The Belongs to Groups section will appear on the screen.
