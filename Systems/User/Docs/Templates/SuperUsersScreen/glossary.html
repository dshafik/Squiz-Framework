Add Existing User Button

This button allows you to add an existing user as a super user.

Super Users Section

This section lists the users that are currently super users.

User Field

This field shows the first name, last name and username for the user.

Email Field

This field shows the email address for the user.

Last Login Field

This field shows the date and time when the user last logged into the system.

Remove User Icon

This icon allows you to remove the user from the Super Users section. This means that the user will no longer be a super user. When you click this icon, the user will remain on the screen but it will be disabled. It will be removed once you click Save. If you marked the wrong user for removal, click the icon once again.
