tasks This article explains how to connect two Squiz Suite products on the Squiz Suite screen.

Adding a Connection

Before you can create a new connection, make sure that the URL of the system has been entered into the URL field on the Squiz Suite screen in both systems.

To add a connection:

  1. Click Add New. Additional fields will appear on the screen.
  2. Enter the URL of the system that you want to connect to in the API URL field.
  3. Click Save.

The connection now needs to be approved in the other system. Once it has been approved, the systems will be connected.
